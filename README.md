# analise-dados-laboratoriais
## Sobre o projeto
Este projeto tem como objetivo aplicar técnicas de análises de dados para explorar dados laboratoriais e gerar indicadores que auxiliem em tomadas de decisões. 

## Objetivos 
- Analisar volume de exames realizados
- Identificar exames mais frequentes
- Criar indicadores de desempenhos
- Avaliar distribuição por setor
- Desenvolver dashboards para visualização de dados

## Ferramentas utilizadas 
- Excel
- SQL
- Python
- Power BI
- GitHub
